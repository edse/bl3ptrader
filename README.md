# bl3ptrader
Bl3p exchange trading bot

## Install
git clone https://github.com/edse/bl3ptrader.git

## Running on Docker
cd bl3ptrader
docker-compose up

## Running on local machine
cd bl3ptrader
make dev

## Requirements
- Python
- Virtualend
- Docker
- InfluxDB
- Graphana
